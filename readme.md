# Movie Nest 🎬

A modern, high-fidelity, and fully responsive replica of the Netflix landing page designed to demonstrate front-end engineering mastery, advanced layout structures, and pixel-perfect design accuracy.

> **💡 Note for Reviewers:** While this project is fully responsive and optimized for mobile viewports, it is highly recommended to view it on a **laptop or desktop screen** to fully appreciate the complex movie-grid background layouts, absolute positioning systems, and cinematic widescreen styling.

---

## 🚀 Project Overview

**Movie Nest** is a front-end showcase project built entirely from scratch. The goal was to recreate the immersive, high-contrast visual interface of Netflix with extreme attention to detail, ranging from custom transparent navigation overlays to interactive components. 

### Key Features
- **Responsive Fluid Design:** Seamlessly transitions across ultra-widescreen monitors, laptops, tablets, and smartphones using fluid percentage-based sizing and modern CSS viewports.
- **Cinematic Visual Grids:** Features a multi-layered parallax hero section using an intricate movie poster grid background blended with absolute gradient overlays.
- **Structured Media Integration:** Embedded HTML5 video elements configured with custom positioning rules to simulate native streaming preview cards.
- **Custom UI Components:** Includes a styled global multi-language selector dropdown, high-fidelity inputs, and sleek hover-interactive FAQ panels.

---

## 🛠️ Tech Stack & Architecture

- **HTML5:** Leveraging semantic elements (`<nav>`, `<section>`, `<footer>`) to construct an accessible, SEO-friendly document tree.
- **CSS3:** Built using custom modular styles, advanced coordinate positioning (`absolute`/`relative`), opacity mixing, aspect-ratio scaling, and fluid text typography.
- **FontAwesome v6.7.2:** Integrated via premium CDN for lightweight, vector-sharp iconography across the interface.
- **Google Fonts:** Utilizing highly optimized typography pairings (**Montserrat** & **Poppins**) to replicate authentic digital-streaming aesthetic frameworks.
